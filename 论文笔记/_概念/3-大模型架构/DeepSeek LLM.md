# DeepSeek LLM

## 定义
DeepSeek LLM 是 DeepSeek-AI 开发的一系列开源大语言模型，采用标准的 Transformer 架构（使用 Multi-Head Attention、RoPE、SwiGLU、GQA 等组件），是 DeepSeekMath、DeepSeek-Coder 等衍生模型的基础。

## 核心要点
1. DeepSeek LLM 系列模型架构使用 Multi-Head Attention、RoPE、SwiGLU、GQA
2. DeepSeekMath 使用 DeepSeek-Coder-Base-v1.5 7B 作为初始化模型
3. DeepSeek LLM 系列遵循"长期主义"（longtermism）的扩展策略

## 代表工作
- [[DeepSeekMath]]: 基于 DeepSeek LLM 架构进行数学领域继续预训练
