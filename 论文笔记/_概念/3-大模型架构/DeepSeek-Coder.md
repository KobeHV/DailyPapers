# DeepSeek-Coder

## 定义
DeepSeek-Coder 是 DeepSeek-AI 开发的代码大语言模型系列，在大量代码数据上预训练，具有强大的编程能力。DeepSeek-Coder-Base-v1.5 7B 被用作 DeepSeekMath 的初始化模型。

## 核心要点
1. DeepSeek-Coder-Base-v1.5 7B 在代码数据上预训练，初始化后继续数学训练
2. 代码训练有益于数学推理，尤其在工具辅助推理场景下
3. 代码与数学混合训练可以缓解灾难性遗忘
