# 累积对数概率 (Cumulative Log-Probability)

## 定义
对序列生成过程中每个 token 的对数条件概率求和，作为该序列质量的评分指标。

## 数学形式
$$
\text{score}(s) = \sum_{t=1}^{|s|} \log P(t_t | t_{<t}, c)
$$

**含义**: 序列的累积对数概率越高，模型认为该序列越合理

**符号说明**:
- $s$: token 序列
- $t_t$: 序列中第 $t$ 个 token
- $t_{<t}$: 第 $t$ 个 token 之前的所有 token
- $c$: 上下文条件

## 核心要点
1. 对数概率可加性使得不同长度序列可以比较
2. 倾向于短序列（长序列通常有更低的累积概率）
3. TreeThink 使用它作为节点评分的基础评估器
4. 常与其他归一化策略配合使用

## 代表工作
- [[TreeThink]]: 将累积对数概率作为核心评估器之一
- [[BFS-Prover]]: 使用长度归一化的累积对数概率
