# Lean 4

## 定义
Lean 4 是一个交互式定理证明器（proof assistant）和函数式编程语言，用于形式化数学证明，是 Lean 定理证明器的最新主要版本。

## 核心要点
1. 基于依赖类型论（Dependent Type Theory）的证明助手
2. 可作为编程语言使用，拥有完善的包管理器（Lake）
3. **Kimina server**: Lean 4 的 REPL 服务器，支持与外部系统交互式通信
4. 广泛应用于形式化数学（如 mathlib4 库）和 ML 驱动的定理证明

## 代表工作
- [[TreeThink]]: 支持通过 Kimina server 与 Lean 4 进行同步和异步通信，实现实时证明验证
