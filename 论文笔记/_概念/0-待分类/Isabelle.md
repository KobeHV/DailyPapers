# Isabelle/HOL

## 定义
Isabelle 是一个通用的交互式定理证明框架，Isabelle/HOL 是其最流行的对象逻辑，基于高阶逻辑（Higher-Order Logic）进行形式化证明。

## 核心要点
1. 与 Lean 4 和 Coq/Rocq 并列为三大主流证明助手
2. 使用高阶逻辑作为基础，拥有成熟的自动化证明策略
3. 广泛应用于形式化验证（如 seL4 微内核验证）
4. TreeThink 计划支持但尚未实现 Isabelle/HOL 集成
